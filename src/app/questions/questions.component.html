HTML (Hypertext Markup Language) is a standard markup language used to create web pages and web applications. HTML is written in the form of HTML elements consisting of tags enclosed in angle brackets. HTML tags are used to describe the structure and content of a web page, and can be used to specify text formatting, links, images, and other features.

Cascading Style Sheets (CSS) is a stylesheet language used for describing the look and formatting of a document written in HTML. CSS is used to control the style and layout of web pages and web applications, and can be used to specify things like colors, fonts, sizes, and positions of elements on a page.

SQL (Structured Query Language) is a standard programming language used to manage and manipulate data stored in relational databases. SQL is used to create, modify, and query databases, and it is used to perform a wide range of tasks, including data definition (creating and modifying tables and other database structures), data manipulation (inserting, updating, and deleting data), and data querying (retrieving data from a database).

Git is a version control system (VCS) that is used to track changes to files and directories, and to collaborate with others on software development projects. Git stores data as a series of snapshots of a project, rather than storing data as a set of differences between files. This allows Git to track the entire history of a project, including who made each change, when they made it, and why they made it.
